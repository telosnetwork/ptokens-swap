# pTokens Migration (ui)

Webapp to help users to redeem pTokens TLOS for OFT TLOS


If haven't install the dev tools yet:
```bash
# setting up tools
nvm install v22.12.0
nvm use v22.12.0
npm i -g yarn
yarn global add @quasar/cli@2.4.1
```

To run the webapp:
```bash
# running the pTokens Webapp UI
cd ui/
yarn
quasar dev
```
